Flash content (Math Sheet)

This page embeds a legacy Flash (.swf) application. Modern browsers no longer include Adobe Flash Player. To view it in Google Chrome, install a Flash-compatible emulator extension (for example Ruffle or another extension that handles Shockwave Flash content), then reload this page.

In Chrome:

  1. Open the Chrome Web Store and add a Flash emulator extension (e.g. Ruffle).
  2. Pin or enable the extension if prompted.
  3. Return to this tab and use Reload (or press Ctrl+Shift+R / Cmd+Shift+R for a hard refresh).

Use a browser that supports a Flash emulator for application/x-shockwave-flash, or switch to Google Chrome and follow the steps above.

If you already installed an emulator but still see this message, the browser may not be exposing the Flash handler yet—try a full restart of Chrome, then reload this page.
